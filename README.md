### @hexlet/code

@hexlet/code is a compilation math games

### Requirements

**Node.js version > 13  is required.**

### Installation

Use the package manager [npm](https://www.npmjs.com/package/package) to install @hexlet/code

```bash
npm install @hexlet/code
```
## Usage

```Node.js
brain-calc

brain-even

brain-gcd

brain-prime

brain-progression
```
